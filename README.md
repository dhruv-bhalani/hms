# 🏥 AuraHealth Hospital Management System

A modern, responsive **Hospital Management System** built using **HTML, Tailwind CSS, and JavaScript**. The project provides an attractive hospital website along with **Patient** and **Admin Portal** interfaces for managing appointments and healthcare services.

---

## 📌 Features

### 🌐 Public Website
- Modern responsive landing page
- Hospital information
- Departments
- Doctors section
- Services
- Gallery
- Patient testimonials
- FAQ section
- Contact form
- Emergency contact section

### 👨‍⚕️ Patient Portal
- Patient Login
- Book Appointments
- View Medical Records
- View Prescriptions
- View Bills
- Secure Patient Dashboard

### 👨‍💼 Admin Portal
- Admin Login
- Dashboard
- Manage Patients
- Manage Doctors
- Manage Appointments
- Analytics Dashboard
- Reports Management

### 🎨 UI Features
- Responsive Design
- Dark Mode
- Smooth Animations
- Glassmorphism UI
- Toast Notifications
- Scroll to Top Button
- Interactive Charts
- Mobile Friendly

---

## 🛠️ Technologies Used

- HTML5
- Tailwind CSS
- JavaScript (ES6)
- Chart.js
- Lucide Icons
- Google Fonts (Inter)
